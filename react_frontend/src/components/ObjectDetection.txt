Ключевые функции компонента:
Управление состоянием:

file - хранит выбранный файл

preview - временный URL для предпросмотра

result - URL обработанного изображения

loading - индикатор загрузки

error - сообщение об ошибке

Основные обработчики:

handleFileChange - вызывается при выборе файла, создает предпросмотр

handleSubmit - отправляет изображение на бэкенд и обрабатывает ответ

Взаимодействие с бэкендом:

Отправка файла через FormData

Обработка успешного ответа (получение Blob)

Обработка ошибок (отображение сообщения)

Пользовательский интерфейс:

Кастомный input для выбора файла

Кнопка с состоянием загрузки

Отображение оригинального и обработанного изображения

Вывод ошибок в понятном формате

Этот компонент представляет собой законченное решение для взаимодействия с вашим FastAPI бэкендом, обеспечивая удобный интерфейс для загрузки изображений и просмотра результатов детекции объектов.
